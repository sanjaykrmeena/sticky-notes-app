# Sticky Notes App
Sticky Notes web application stores text composed by the users forever so the client can admittance to those notes and can 
include new notes/messages whenever.

## Features
* Write short Notes
* Delete Notes
* Update Notes

## Tech Stack
* HTML
* CSS 
* JavaScript
* Local Storage
